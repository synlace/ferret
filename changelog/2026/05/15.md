# [2026-05-15]

### Added
- feat(setup): first-run setup wizard for AI provider configuration
  - New `GET/POST/DELETE /api/setup` and `POST /api/setup/test` endpoints
  - Supports OpenRouter, OpenAI, Anthropic, Gemini, DeepSeek, Mistral, Ollama, LM Studio
  - OpenRouter supports both a regular API key and an optional provisioning key for per-project sub-key creation
  - Dynamic AI config loaded from DB at startup via `deps.reload_ai_config()`; falls back to legacy `OPENROUTER_PROVISIONING_KEY` env var for existing deployments
  - "Skip for now" option bypasses wizard without blocking access
  - Frontend wizard at `/setup` (4-step: Provider → Configure → Model → Done) with LobeHub colour brand icons
  - `AppShell` redirects to `/setup` on first load if setup is not complete; setup page renders without sidebar
  - Settings page shows active AI provider/model and a "Re-run setup wizard" button
